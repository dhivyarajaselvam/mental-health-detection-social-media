* Mental Health Detection from Social Media Data using Naive Bayes and Random Forest

An interpretable machine learning project for detecting mental health conditions from social media data using Naive Bayes and Random Forest, with a comparative analysis of predictive performance and model interpretability.

* Project Overview

Mental health has become an increasingly important area of research, especially with the widespread use of social media. This project aims to identify potential mental health conditions from social media data using supervised machine learning techniques. The study compares **Naive Bayes** and **Random Forest** classifiers to evaluate their effectiveness in terms of accuracy and overall classification performance.

* Objectives

- Detect mental health conditions from social media data.
- Compare the performance of Naive Bayes and Random Forest.
- Evaluate models using standard classification metrics.
- Improve understanding of interpretable machine learning techniques.

* PICO Framework

Problem - Mental Health Detection from Social Media Data 
Intervention - Naive Bayes 
Comparison - Random Forest 
Outcome - Improved predictive accuracy and interpretable classification 
Groups - 2 
Sample Size per Group - 325 
Total Sample Size - 650
